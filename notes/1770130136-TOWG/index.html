关于 Go 的一些点

2026-02-052026-07-07
CSGO

Go语言的亮点是什么：简单、内置对并发的支持、工程效率>语言炫技（快速编译、强大统一的标准库、内置工具链）

  1. 语法简单、类型系统的克制；代码容读易维护
  2. 内置的并发支持：不要通过共享内存通信，而是通过通信共享内存
  • goroutine 用户态调度的任务
  • channel goroutine直接通信的方式
  • CSP模型
  1. 健壮标准库：
  • errors 包：错误是值
  • io 包：对读写的抽象
  • context 包：请求链上节点对取消通知的感知
  • sync 包：提供并发原语
  • 其他如os包，http包
  1. 官方支持的工具链：go build, go test, go tool trace, go tool pprof 等

标准库学习

标准库是随语言一起发布和维护的工具库，基于语言特性构建，安装即可用，且与语言版本共同演进、遵循相同的兼容性承诺。

核心语义包：定义协议（接口）或提供原语

IO，与操作系统（文件、进程）的交互：

  • bufio 包，为io.Writer,io.Reader增加了缓冲区
  • os 包，语言如何与操作系统交互

交互的数据如何处理：

  • strings包，bytes包，基础数据类型
  • slices包，maps包，复合数据类型

网络：

  • net包，http包，网络相关
