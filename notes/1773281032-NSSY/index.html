  1. S — Situation：先交代背景和现状
  2. C — Complication：指出在这个背景下出现了什么问题或张力
  3. Q — Question：自然引出“那现在该怎么办”
  4. A — Answer：给出简明直接的回答或方案

总结：现在是什么情况 -> 出了什么问题 -> 所以我们要回答什么 -> 我的建议是什么

SCQA和金字塔原则的关系：SCQA用于引出问题，金字塔原则用于组织回答；还有MECE原则，保证回答的结构不乱。
