zig: no hidden control flow

  • 编译期 hidden：
    • 类型推导、泛型实例化、trait 解析、隐式转换、宏展开、编译器优化。
  • 运行时 hidden：
    • 内存分配、资源释放、异常跳转、动态分发、装箱、GC、调度、锁、IO。
Hidden flow 类型 其他语言常见形式 Zig 的做法
隐藏错误跳转 exception try / catch 显式
隐藏分配 append/new/string concat 传 allocator
隐藏释放 GC / destructor defer + 显式 cleanup
隐藏动态分发 interface / virtual method 手写 vtable 或 anytype 静态分发
隐藏类型擦除 any / Object / interface boxing *anyopaque 显式
隐藏用户代码 operator overloading / copy ctor 不支持这类魔法
隐藏 runtime GC / green thread / async runtime runtime 尽量显式
