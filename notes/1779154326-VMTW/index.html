Go 标准库的结构化日志包，记录由 time、level、message 和 key-value 属性组成。

核心架构 Logger -> Record -> Handler

  • Logger：暴露 Debug/Info/Warn/Error 等方法，每条调用生成 Record
  • Handler：接收 Record 决定输出方式；内置 TextHandler（key=value）和 JSONHandler（行分隔 JSON）

SetDefault 让顶层函数和 log 包都走自定义 handle

  • 不用全局 logger，把 logger 作为依赖传入：测试、库代码、多环境配置都更可控
  • 使用 LogAttrs ：字段类型更明确，不用松散 kv
  • 使用 slog.Attr 构造器：封装业务字段 helper
  • 复杂对象用 slog.Group
  • 用 sloglint 强制规则
  1. 公共属性用 With 预格式化
  2. 参数总会求值——昂贵计算用 LogValuer 延迟
  3. 用 LogAttrs 避免 any 装箱
  4. Context 版本：可传 ctx，便于 handler 提取 trace 信息
