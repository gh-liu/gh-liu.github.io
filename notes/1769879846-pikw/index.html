第一部分：基础概念

什么是 Reed-Kellogg 图解法

Reed-Kellogg 图解法（Reed-Kellogg Sentence Diagramming）是由 Alonzo Reed 和 Brainerd Kellogg 在 1877 年创立的一种句子分析方法。它通过水平线、垂直线和斜线的组合，将句子的语法结构可视化，帮助学习者：

  • 识别句子的核心成分（主语、谓语、宾语）
  • 理解修饰语与被修饰成分的关系
  • 分析从句在句子中的功能
  • 发现并纠正语法错误

基本绘图规则

符号用途示例
水平基线 ────放置句子的核心成分主语、谓语、宾语
穿过基线的竖线 │分隔主语和谓语Subject │ Verb
不穿过基线的竖线 ┃分隔谓语和直接宾语Verb ┃ Object
斜线 ╲分隔系动词和表语（指向主语）is ╲ happy
下挂斜线 ╲放置修饰语（形容词、副词）在被修饰词下方
弯曲支架线介词短语prep + 宾语
虚线 - - -连接并列成分and, or, but
支架/底座从句整个从句作为句子成分

与五种基础句型的对应

回顾英语五种基础句型及其图解形式：

句型结构图解特征
SV主语 + 不及物动词Subject │ Verb
SVO主语 + 及物动词 + 宾语Subject │ Verb ┃ Object
SVC主语 + 系动词 + 表语Subject │ is ╲ Complement
SVOO主语 + 动词 + 间接宾语 + 直接宾语间接宾语在下方
SVOC主语 + 动词 + 宾语 + 宾补宾补用斜线连接宾语

第二部分：核心句子结构

1. 主语 + 谓语 (SV)

最简单的句子结构：主语执行一个动作（不及物动词）。

例句: Birds fly.

        Birds │ fly
              │

绘图步骤：

  1. 画一条水平基线
  2. 在基线中央画一条穿过基线的垂直线
  3. 主语 “Birds” 写在左边
  4. 谓语 “fly” 写在右边

更多例句：

The baby sleeps.

      baby │ sleeps
       ╲   │
       The │

注意：冠词 “The” 作为限定词，也放在修饰语的位置（斜线下方）。

2. 主语 + 谓语 + 宾语 (SVO)

及物动词需要一个宾语来接受动作。

例句: The cat caught the mouse.

       cat    │  caught  │ mouse
        ╲     │          │   ╲
        The   │          │   the

绘图步骤：

  1. 画水平基线
  2. 主谓分隔线穿过基线
  3. 谓语和宾语之间的分隔线不穿过基线（这是区别 SV 和 SVO 的关键！）
  4. 修饰语（冠词）放在斜线下方

例句: She reads books every day.

        She  │  reads   │ books
             │     ╲    │
             │   every  │
             │     ╲    │
             │     day  │

等等，这个图解有问题。“every day” 是时间状语，修饰动词 “reads”，应该这样画：

        She  │  reads   │ books
             │    ╲     │
             │    day   │
             │     ╲    │
             │    every │

3. 主语 + 系动词 + 表语 (SVC)

系动词（linking verb）连接主语和表语，表语可以是名词或形容词。

例句: The flower is beautiful.

      flower │    is    ╲ beautiful
        ╲    │
        The  │

关键点：表语后的分隔线是斜线，指向主语，表示主语和表语之间的”等价”关系。

例句: He became a doctor.

         He  │  became  ╲ doctor
             │              ╲
             │               a

常见系动词: be, become, seem, appear, feel, look, sound, taste, smell, remain, stay

4. 主语 + 谓语 + 间接宾语 + 直接宾语 (SVOO)

双宾语结构：动词有两个宾语，间接宾语（通常是人）和直接宾语（通常是物）。

例句: She gave him a book.

        She  │   gave   │ book
             │     ╲    │   ╲
             │      ╲   │    a
             │       ╲  │
             │       him│

另一种更标准的画法：

        She  │   gave   │ book
             │    │     │   ╲
             │   him    │    a

间接宾语 “him” 放在动词下方，用竖线连接。

例句: The teacher taught us English.

    teacher  │  taught  │ English
       ╲     │    │     │
       The   │   us     │

5. 主语 + 谓语 + 宾语 + 宾补 (SVOC)

宾语补语补充说明宾语的状态或身份。

例句: They elected him president.

       They  │ elected  │ him  ╲ president
             │          │

例句: The news made her happy.

       news  │   made   │ her  ╲ happy
        ╲    │          │
        The  │          │

绘图要点：宾语和宾补之间用斜线分隔，斜线指向宾语，表示补语说明宾语。

             │

             │          │

这是 SVO 结构，主语 “I”，及物动词 “love”，宾语 “music”。注意谓语和宾语之间的竖线不穿过基线。

点击查看答案 
        She  │    is    ╲ teacher
             │              ╲

第三部分：修饰语

形容词 Adjectives

形容词修饰名词，在图解中放在被修饰名词的下方斜线上。

例句: The big dog barked loudly.

        dog   │   barked
         ╲    │      ╲
         big  │    loudly
          ╲   │
          The │

多个形容词修饰同一名词：

例句: A small red ball rolled away.

       ball   │   rolled
        ╲     │      ╲
       red    │    away
        ╲     │
      small   │
        ╲     │
         A    │

所有修饰语都挂在被修饰词下方。

副词 Adverbs

副词修饰动词、形容词或其他副词，同样放在被修饰词的下方斜线上。

例句: She sings beautifully.

        She  │   sings
             │      ╲
             │   beautifully

副词修饰形容词：

例句: He is very tall.

         He  │    is    ╲ tall
             │              ╲
             │           very

“very” 修饰形容词 “tall”，所以挂在 “tall” 下面。

副词修饰副词：

例句: She runs quite quickly.

        She  │   runs
             │     ╲
             │   quickly
             │      ╲
             │     quite

介词短语 Prepositional Phrases

介词短语由介词 + 宾语组成，整体作为修饰语。用特殊的弯曲支架表示。

例句: The book on the table is mine.

       book   │    is    ╲ mine
         ╲    │
         The  │
          ╲───┼──────────────
          on  │  table
              │    ╲
              │    the

更标准的画法：

       book      │    is    ╲ mine
         ╲       │
         The     │
             ╲   │
              on │
               ╲ │
             table
                ╲
                the

介词短语结构：

            ╲
         prep
            ╲
          object
             ╲
          modifier

例句: He lives in a small house.

         He  │  lives
             │    ╲
             │     in
             │       ╲
             │      house
             │         ╲
             │        small
             │           ╲
             │            a

介词短语修饰名词（作定语）vs 修饰动词（作状语）：

  • The man with glasses reads. → 修饰名词 “man”
  • He reads with glasses. → 修饰动词 “reads”

第四部分：复杂结构

并列结构 Compound Structures

当句子有并列成分时，使用虚线连接。

并列主语：

例句: Tom and Jerry are friends.

        Tom
           ╲
            ╲
         and─────┤   are   ╲ friends
            ╱    │
           ╱     │
       Jerry     │

另一种表示法：

        Tom ─┐
             ├───│   are   ╲ friends
       Jerry ─┘  │
         and     │

并列谓语：

例句: She sings and dances.

                   ┌─ sings
        She  │ ────┤
                   └─ dances
                 and

并列宾语：

例句: I like apples and oranges.

          I  │  like   │ apples ─┐
             │         │         ├── and
             │         │ oranges ─┘

定语从句 Adjective Clauses

定语从句修饰名词，整个从句放在被修饰名词下方的支架上。

例句: The man who called you is my father.

        man       │    is    ╲ father
         ╲        │              ╲
         The      │              my
          ║       │
          ║ who │ called │ you
          ╚══════════════════════

从句 “who called you” 作为整体修饰 “man”。

用虚线连接关系词和先行词的画法：

        man       │    is    ╲ father
         ╲        │              ╲
         The      │              my
          :       │
          :       │
       ───────────────────
       who │ called │ you
           │        │

例句: The book that I bought is interesting.

       book       │    is    ╲ interesting
         ╲        │
         The      │
          :       │
       ───────────────────
        I │ bought │ that
          │        │

注意：关系代词 “that” 在从句中作宾语，所以在从句图解中放在宾语位置。

状语从句 Adverb Clauses

状语从句修饰动词，放在主句动词下方。

例句: I will go if it stops raining.

          I  │ will go
             │    ╲
             │     if
             │    ═══════════════
             │     it │ stops │ raining
             │        │       │

从句用支架表示，从属连词（if, when, because 等）标在支架开头。

例句: Because he was tired, he went to bed early.

         he  │   went
             │     ╲
             │   early
             │      ╲
             │       to
             │        ╲
             │       bed
             │
             │    Because
             │    ════════════════
             │     he │ was ╲ tired
             │        │

名词从句 Noun Clauses

名词从句可以作主语、宾语或表语，放在支架上作为整体。

作宾语：

例句: I know that he is honest.

          I  │  know   │ [从句]
             │         │
             │         ╚══════════════════
             │          that
             │          ────────────────
             │          he │ is ╲ honest
             │             │

作主语：

例句: What you said is true.

     [从句]   │    is    ╲ true
       ║      │
    ═══════════════
    you │ said │ What
        │      │

整个从句 “What you said” 作为主语。

第五部分：综合练习

以下是综合难度的练习，测试你对各种句子结构的理解。

练习 1: “The clever student quickly solved the difficult problem.”

点击查看答案 
     student  │  solved   │ problem
        ╲     │     ╲     │    ╲
      clever  │  quickly  │  difficult
        ╲     │           │     ╲
        The   │           │    the
  • 主语：student（被 clever, The 修饰）
  • 谓语：solved（被 quickly 修饰）
  • 宾语：problem（被 difficult, the 修饰）

练习 2: “My friend gave me a beautiful birthday gift.”

点击查看答案 
     friend   │   gave   │   gift
        ╲     │    │     │     ╲
        My    │   me     │  birthday
              │          │     ╲
              │          │  beautiful
              │          │     ╲
              │          │     a
  • SVOO 结构
  • 间接宾语 “me” 在动词下方
  • 直接宾语 “gift” 被 a, beautiful, birthday 修饰

练习 3: “The man standing at the door is my uncle.”

点击查看答案 
        man       │    is    ╲ uncle
         ╲        │              ╲
         The      │              my
          ╲       │
       standing   │
            ╲     │
            at    │
             ╲    │
           door   │
              ╲   │
              the │
  • “standing at the door” 是现在分词短语，修饰 “man”
  • 介词短语 “at the door” 修饰 “standing”

练习 4: “She told me that she would come if she had time.”

点击查看答案 
        She  │   told   │ [名词从句]
             │    │     │
             │   me     │
             │          ╚══════════════════════════
             │          that
             │          ─────────────────────────
             │          she │ would come
             │              │     ╲
             │              │     if
             │              │    ════════════════
             │              │    she │ had │ time
             │              │        │     │
  • SVOO 结构，间接宾语 “me”，直接宾语是名词从句
  • 名词从句内部包含状语从句 “if she had time”

练习 5: “What he said and what he did were completely different.”

点击查看答案 
    [从句1] ──┐
              ├──│   were   ╲ different
    [从句2] ──┘  │              ╲
          and    │          completely

    从句1: he │ said │ What
              │      │

    从句2: he │ did │ What
              │     │
  • 并列主语：两个名词从句
  • 表语 “different” 被副词 “completely” 修饰

学习建议

  1. 从简单开始：先熟练掌握 SV、SVO、SVC 三种基本结构
  2. 多动手画：纸笔练习比单纯阅读更有效
  3. 分析真实句子：尝试图解你阅读中遇到的句子
  4. 逐步增加复杂度：修饰语 → 介词短语 → 从句
  5. 对照中文理解：结合你的语法笔记，理解句子成分对应关系
