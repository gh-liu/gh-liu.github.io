基于 io.Writer/io.Reader 的缓存适配器，增加了缓冲区 buffer

Go: 标准库 io包定义了IO操作的读写关闭设置偏移的接口，bufio 是基于 io.Reader/io.Writer 的一层缓冲适配器，主要增加用户态 buffer，从而减少底层 Read/Write 调用次数，并提供一些更方便的读写方法。

Reader

  1. 从底层 reader 多读一点，比如4kb
  2. 后续读取从 bufio 内部 buffer 里取
  3. buffer 用完后，再向底层 reader 批量读取

更方便的方法

Writer

  1. 先写到内部 buffer
  2. buffer 满了，或主动 Flush，才一次性写到底层 writer

ReadWriter

把 bufio.Reader 和 bufio.Writer 组合起来 Reader, Writer 有各自的buffer

Scanner

高层、方便的 token 迭代器
