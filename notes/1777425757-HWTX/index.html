Go: web framework

https://blog.jetbrains.com/go/2026/04/28/popular-golang-web-frameworks

  1. 标准库
  • 提供了 HTTP server、handler、middleware 组合、路由等能力
  • 缺点：业务复杂后，路由、中间件、参数绑定、错误处理等样板代码会变多
  1. chi
  • 使用标准的 http.Handler / http.HandlerFunc 风格，middleware 兼容性好，外部依赖少
  1. gin
  • 最流行、上手快、社区最大
  1. echo
  • 和 Gin 类似，也是基于 net/http，但它的 handler 返回 error
  1. fiber
  • 不是基于 net/http，而是基于 fasthttp
  • 性能很强，但和 Go 标准 Web 生态兼容性更弱

gin vs. echo

Gin：Context 驱动 Echo：Error-return 驱动
