幺半群 Monoid（来自抽象代数）

2026-04-082026-04-081 引用
CS

最简单的代数结构，三要素：

  1. 一个集合 S
  2. 一个二元运算 ·（把两个元素合并成一个）
  3. 一个单位元 e（合并后等于没合并）

满足两条法则：

  • 结合律：(a · b) · c = a · (b · c)
  • 单位律：e · a = a 且 a · e = a
集合 运算 单位元
整数 + 0
整数 * 1
字符串 拼接 空串
布尔 && True
函数 a → a 组合 ∘ id

直觉：可以不断合并、合并顺序无所谓、有一个空操作的东西。
