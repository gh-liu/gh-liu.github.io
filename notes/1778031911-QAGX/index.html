Go: map swiss table

https://go.dev/blog/swisstable

本质：一套 cache-friendly + SIMD + bit trick 的实现方式；极致利用 CPU cache 和向量指令

现代 CPU：算术便宜，内存贵；慢在cachemiss；用更多的bittrick，换更少的内存访问（减少miss）。

  • open addressing（开放寻址）
  • metadata（控制字节）分离
  • SIMD 批量匹配
  • 减少随机内存访问
