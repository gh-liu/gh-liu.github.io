Kafka 4.x й»ҳи®Өе·Із»Ҹиҝӣе…Ҙж—  ZooKeeper зҡ„ KRaft ж—¶д»ЈпјҢеҚ•жңәеӯҰд№ ж—¶дёҚйңҖиҰҒеҶҚеҚ•зӢ¬иө· ZooKeeper

S д»Җд№ҲеңәжҷҜ

еӯҰд№  Kafka ж–°зүҲжң¬йғЁзҪІгҖҒйҳ…иҜ» Kafka жһ¶жһ„иө„ж–ҷпјҢжҲ–иҖ…жҺ’жҹҘйӣҶзҫӨжҺ§еҲ¶йқўй—®йўҳж—¶пјҢз»ҸеёёдјҡзңӢеҲ° KRaftгҖӮ

иҝҮеҺ» Kafka дҫқиө– ZooKeeper дҝқеӯҳйӣҶзҫӨе…ғж•°жҚ®пјҢжҜ”еҰӮпјҡ

  • broker жіЁеҶҢдҝЎжҒҜ
  • topic / partition е…ғж•°жҚ®
  • leader йҖүдёҫзӣёе…ізҠ¶жҖҒ
  • ACLгҖҒй…ҚйўқзӯүйғЁеҲҶжҺ§еҲ¶йқўдҝЎжҒҜ

зҺ°еңЁ Kafka жӯЈеңЁз”Ё KRaft жӣҝд»Ј ZooKeeperпјҢиҝҷе·Із»ҸжҳҜзҗҶи§Ј Kafka ж–°жһ¶жһ„ж—¶з»•дёҚиҝҮеҺ»зҡ„дёҖйғЁеҲҶгҖӮ

C д»Җд№ҲеҶІзӘҒ

Kafka жң¬иә«е·Із»ҸжҳҜдёҖдёӘеҲҶеёғејҸж—Ҙеҝ—зі»з»ҹпјҢдҪҶе®ғиҝҮеҺ»еҸҲдҫқиө–еҸҰдёҖдёӘеҲҶеёғејҸзі»з»ҹ ZooKeeper жқҘдҝқеӯҳиҮӘе·ұзҡ„е…ғж•°жҚ®пјҢиҝҷдјҡеёҰжқҘеҮ дёӘжҳҺжҳҫеҶІзӘҒпјҡ

  • жһ¶жһ„дёҠеӨҡдёҖеҘ—еӨ–йғЁдҫқиө–пјҢйғЁзҪІжӣҙеӨҚжқӮ
  • иҝҗз»ҙж—¶йңҖиҰҒеҗҢж—¶з»ҙжҠӨ Kafka е’Ң ZooKeeper дёӨдёӘйӣҶзҫӨ
  • ж•…йҡңе®ҡдҪҚи·ЁдёӨеҘ—зі»з»ҹпјҢжҺ’жҹҘи·Ҝеҫ„жӣҙй•ҝ
  • е…ғж•°жҚ®дёҖиҮҙжҖ§гҖҒжҺ§еҲ¶еҷЁйҖүдёҫйҖ»иҫ‘еҲҶж•ЈеңЁдёӨз§ҚжңәеҲ¶йҮҢпјҢзҗҶи§ЈжҲҗжң¬й«ҳ

жң¬иҙЁеҶІзӘҒжҳҜпјҡKafka зҡ„ж•°жҚ®йқўе·Із»ҸеҫҲејәпјҢдҪҶжҺ§еҲ¶йқўй•ҝжңҹдҫқиө–еӨ–йғЁзі»з»ҹгҖӮ

Q д»Җд№Ҳй—®йўҳ

1. KRaft еҲ°еә•жҳҜд»Җд№Ҳ

KRaft = Kafka Raft Metadata mode

е®ғиЎЁзӨә Kafka дёҚеҶҚдҫқиө– ZooKeeperпјҢиҖҢжҳҜдҪҝз”Ё Kafka иҮӘе·ұеҹәдәҺ Raft зҡ„жңәеҲ¶жқҘз®ЎзҗҶйӣҶзҫӨе…ғж•°жҚ®гҖӮ

еҸҜд»ҘжҠҠе®ғзҗҶи§ЈжҲҗпјҡжҠҠеҺҹжқҘж”ҫеңЁ ZooKeeper йҮҢзҡ„жҺ§еҲ¶йқўе…ғж•°жҚ®пјҢж”¶еӣһеҲ° Kafka иҮӘе·ұеҶ…йғЁз®ЎзҗҶгҖӮ

2. е®ғи§ЈеҶідәҶд»Җд№Ҳй—®йўҳ

дё»иҰҒи§ЈеҶідёүзұ»й—®йўҳпјҡ

  • йҷҚдҪҺйғЁзҪІе’Ңиҝҗз»ҙеӨҚжқӮеәҰ
  • з»ҹдёҖе…ғж•°жҚ®еӯҳеӮЁе’ҢжҺ§еҲ¶еҷЁйҖүдёҫжңәеҲ¶
  • и®© Kafka жһ¶жһ„жӣҙе®Ңж•ҙпјҢеҮҸе°‘еҜ№еӨ–йғЁзі»з»ҹзҡ„ејәдҫқиө–

3. KRaft йҮҢзҡ„е…ғж•°жҚ®ж”ҫе“Ә

ж”ҫеңЁ Kafka йӣҶзҫӨиҮӘе·ұзҡ„ metadata log дёӯгҖӮ

иҝҷжҳҜдёҖд»Ҫдё“й—Ёи®°еҪ•йӣҶзҫӨе…ғж•°жҚ®еҸҳжӣҙзҡ„ж—Ҙеҝ—пјҢеҢ…еҗ«пјҡ

  • broker жіЁеҶҢдёҺдёӢзәҝ
  • topic еҲӣе»әгҖҒеҲ йҷӨгҖҒй…ҚзҪ®еҸҳжӣҙ
  • partition еҲҶй…Қ
  • ISRгҖҒleader еҸҳжӣҙзӯүжҺ§еҲ¶йқўдәӢд»¶

е’Ңжҷ®йҖҡдёҡеҠЎж¶ҲжҒҜж—Ҙеҝ—дёҚеҗҢпјҢmetadata log дёҚжҳҜз»ҷдёҡеҠЎж¶Ҳиҙ№зҡ„пјҢиҖҢжҳҜз»ҷ Kafka еҶ…йғЁжҺ§еҲ¶еҷЁе’Ң broker еҗҢжӯҘйӣҶзҫӨзҠ¶жҖҒз”Ёзҡ„гҖӮ

4. KRaft дёӢиҝҳжңү controller еҗ—

жңүпјҢиҖҢдё” controller зҡ„жҰӮеҝөжӣҙйҮҚиҰҒдәҶгҖӮ

еңЁ KRaft жЁЎејҸдёӢпјҢcontroller иҠӮзӮ№иҙҹиҙЈпјҡ

  • жҺҘж”¶е’ҢжҸҗдәӨе…ғж•°жҚ®еҸҳжӣҙ
  • з»ҙжҠӨйӣҶзҫӨе…ғж•°жҚ®зҡ„жңҖж–°зҠ¶жҖҒ
  • еӨ„зҗҶ leader йҖүдёҫгҖҒbroker зҠ¶жҖҒеҸҳеҢ–зӯүжҺ§еҲ¶йҖ»иҫ‘

иҝҷдәӣ controller иҠӮзӮ№дјҡз»„жҲҗдёҖдёӘ quorumпјҢйҖҡиҝҮ Raft йҖүеҮә leader controllerгҖӮ

еҸӘжңү leader controller жүҚиғҪжҸҗдәӨж–°зҡ„е…ғж•°жҚ®и®°еҪ•пјҢе…¶д»– controller иҙҹиҙЈеӨҚеҲ¶е’Ңе®№й”ҷгҖӮ

5. broker е’Ң controller жҳҜд»Җд№Ҳе…ізі»

KRaft жЁЎејҸдёӢпјҢиҠӮзӮ№и§’иүІеҸҜд»ҘжӢҶеҲҶпјҡ

  • brokerпјҡиҙҹиҙЈеӨ„зҗҶз”ҹдә§гҖҒж¶Ҳиҙ№гҖҒеӯҳеӮЁдёҡеҠЎж•°жҚ®
  • controllerпјҡиҙҹиҙЈз®ЎзҗҶе…ғж•°жҚ®е’ҢжҺ§еҲ¶йқўеҶізӯ–

йғЁзҪІдёҠеёёи§ҒдёӨз§Қж–№ејҸпјҡ

  • ејҖеҸ‘ / жөӢиҜ•зҺҜеўғпјҡеҗҢдёҖдёӘиҠӮзӮ№еҗҢж—¶жүҝжӢ… broker,controller
  • з”ҹдә§зҺҜеўғпјҡйҖҡеёёе»әи®®жҠҠ controller quorum зӢ¬з«ӢеҮәжқҘ

иҝҷж ·еҒҡзҡ„зӣ®зҡ„пјҢжҳҜжҠҠж•°жҚ®йқўе’ҢжҺ§еҲ¶йқўиҒҢиҙЈеҲҶзҰ»пјҢйҷҚдҪҺзӣёдә’еҪұе“ҚгҖӮ

6. дёәд»Җд№ҲиҜҙе®ғеҹәдәҺ Raft

еӣ дёә KRaft з”Ёзҡ„жҳҜ Kafka иҮӘе·ұе®һзҺ°зҡ„ Raft еҚҸи®®жқҘеӨҚеҲ¶е…ғж•°жҚ®ж—Ҙеҝ—гҖӮ

е…ій”®зӮ№жҳҜпјҡ

  • controller д№Ӣй—ҙйҖҡиҝҮ Raft дҝқиҜҒе…ғж•°жҚ®еӨҚеҲ¶
  • еӨҡж•°жҙҫзЎ®и®ӨеҗҺпјҢе…ғж•°жҚ®и®°еҪ•жүҚз®—жҸҗдәӨ
  • leader controller е®•жңәеҗҺпјҢеү©дҪҷ controller еҸҜд»ҘйҮҚж–°йҖүдё»

жүҖд»Ҙ KRaft зҡ„ж ёеҝғдёҚжҳҜеҺ»жҺү ZooKeeperиҝҷд№Ҳз®ҖеҚ•пјҢиҖҢжҳҜпјҡKafka з”ЁиҮӘе·ұзҡ„е…ұиҜҶеҚҸи®®жҺҘз®ЎдәҶе…ғж•°жҚ®з®ЎзҗҶгҖӮ

A еӣһзӯ”й—®йўҳ

дёҖеҸҘиҜқзҗҶи§ЈпјҡKRaft жҳҜ Kafka з”Ё Raft еҚҸи®®еҶ…е»әе…ғж•°жҚ®з®ЎзҗҶиғҪеҠӣгҖҒд»ҺиҖҢжӣҝд»Ј ZooKeeper зҡ„ж–°жһ¶жһ„жЁЎејҸгҖӮ

дёәд»Җд№Ҳ Kafka иҰҒеј•е…Ҙ KRaft

еӣ дёәж—§жһ¶жһ„дёӢ Kafka дҫқиө– ZooKeeper з®ЎзҗҶе…ғж•°жҚ®пјҢеҜјиҮҙпјҡ

  • зі»з»ҹз»„д»¶жӣҙеӨҡ
  • йғЁзҪІдёҺеҚҮзә§жӣҙйә»зғҰ
  • ж•…йҡңжҺ’жҹҘй“ҫи·Ҝжӣҙй•ҝ
  • жҺ§еҲ¶йқўдёҚеӨҹз»ҹдёҖ

KRaft зҡ„зӣ®ж Үе°ұжҳҜжҠҠиҝҷдәӣиғҪеҠӣж”¶еӣһ Kafka еҶ…йғЁгҖӮ

KRaft еёҰжқҘзҡ„зӣҙжҺҘж”¶зӣҠ

  • е°‘з»ҙжҠӨдёҖеҘ— ZooKeeper йӣҶзҫӨ
  • е…ғж•°жҚ®з®ЎзҗҶйҖ»иҫ‘жӣҙз»ҹдёҖ
  • жҺ§еҲ¶еҷЁйҖүдёҫжңәеҲ¶жӣҙжё…жҷ°
  • ж•ҙдҪ“жһ¶жһ„жӣҙз®ҖеҢ–
  • жӣҙз¬ҰеҗҲ Kafka еҗҺз»ӯжј”иҝӣж–№еҗ‘

йңҖиҰҒи®°дҪҸзҡ„е…ій”®иҜҚ

  • KRaft = Kafka Raft
  • metadata log
  • controller quorum
  • leader controller
  • broker / controller roles
  • ZooKeeper removal

е®№жҳ“ж··ж·Ҷзҡ„зӮ№

KRaft дёҚжҳҜз”ЁжқҘеӯҳдёҡеҠЎж¶ҲжҒҜзҡ„

е®ғз®ЎзҗҶзҡ„жҳҜ йӣҶзҫӨе…ғж•°жҚ®пјҢдёҚжҳҜ topic йҮҢзҡ„жҷ®йҖҡдёҡеҠЎж¶ҲжҒҜгҖӮ

KRaft дёҚжҳҜжҠҠ ZooKeeper еҠҹиғҪеҺҹе°ҒдёҚеҠЁжҗ¬иҝӣ Kafka

е®ғдёҚжҳҜз®ҖеҚ•еҶ…еөҢдёҖдёӘ ZooKeeperпјҢиҖҢжҳҜжҚўжҲҗдәҶ Kafka иҮӘе·ұзҡ„е…ғж•°жҚ®ж—Ҙеҝ—е’Ң Raft е…ұиҜҶжңәеҲ¶гҖӮ

еҺ»жҺү ZooKeeper дёҚд»ЈиЎЁжІЎжңү controller

controller дҫқз„¶еӯҳеңЁпјҢеҸӘжҳҜе®ғзҡ„е·ҘдҪңж–№ејҸе’Ңе…ғж•°жҚ®еӯҳеӮЁж–№ејҸеҸҳдәҶгҖӮ

иҝҒз§»е’Ңе®һи·өжіЁж„ҸзӮ№

  • иҖҒзүҲжң¬ Kafka еҸҜиғҪд»Қз„¶иҝҗиЎҢеңЁ ZooKeeper жЁЎејҸпјҢиҜ»ж–ҮжЎЈж—¶иҰҒе…ҲеҢәеҲҶйғЁзҪІжЁЎејҸ
  • зңӢеҲ° process.roles=broker,controller иҝҷзұ»й…ҚзҪ®ж—¶пјҢеҹәжң¬е°ұжҳҜ KRaft жЁЎејҸ
  • зңӢеҲ° controller.quorum.voters ж—¶пјҢиҜҙжҳҺйңҖиҰҒжҳҫејҸй…ҚзҪ® controller quorum
  • KRaft йӣҶзҫӨеҗҜеҠЁеүҚйҖҡеёёйңҖиҰҒе…Ҳж јејҸеҢ–е…ғж•°жҚ®зӣ®еҪ•
  • з”ҹдә§зҺҜеўғйҮҚзӮ№е…іжіЁ controller quorum зҡ„еҘҮж•°иҠӮзӮ№йғЁзҪІдёҺе®№й”ҷи®ҫи®Ў

еҸҜд»ҘжҖҺд№Ҳи®°

еҸҜд»Ҙз”ЁдёҖеҸҘйЎәеҸЈзҡ„иҜқи®°пјҡ

д»ҘеүҚ Kafka жҠҠи°ҒжқҘз®ЎйӣҶзҫӨдәӨз»ҷ ZooKeeperпјҢзҺ°еңЁ Kafka иҮӘе·ұз”Ё Raft жқҘз®ЎгҖӮ
